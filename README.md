## MBTI 성향 간단 체크 서비스

이 폴더에는 브라우저에서 바로 열어 사용할 수 있는 **간단한 MBTI 성향 체크 웹 페이지**가 들어 있습니다.

- **기술 스택**: 순수 HTML/CSS/Vanilla JS (빌드 도구 없음)
- **실행 방법**
  - 방법 1: `index.html` 파일을 더블 클릭해서 브라우저(Chrome 등)로 엽니다.
  - 방법 2: 간단한 정적 서버(예: VSCode Live Server, `python -m http.server` 등)를 사용해 `MBTI` 폴더를 서빙합니다.

별도의 패키지 의존성은 없으며, `package.json`은 프로젝트 메타데이터 용도로만 존재합니다.
